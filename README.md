# AI-CHATBOT
AI College Enquiry Chatbot (Python)

An AI-based web chatbot developed using Python (Flask) that answers common college-related queries such as admissions, departments, timings, and placements.
The chatbot supports text and voice input with a simple and user-friendly web interface.
